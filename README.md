## Purpose
This tool is designed for:
- Last-minute exam review before the LFW111 certification
- Quick concept lookup without sifting through documentation
- Memorizing key patterns that frequently appear on the exam
- Understanding the 20% of Node.js that covers 80% of exam questions

## Features
- **Interactive Flashcards** - Hover to flip and reveal answers
- **Essential Code Examples** - Minimal code you actually need to know
- **Exam Strategy Tips** - How to approach different question types
- **Quick Reference Tables** - Commands, modules, and concepts at a glance
- **Mobile Responsive** - Study on any device
- **Dark Mode** - Easier on the eyes during late-night study sessions
- **Printable** - Generate a physical cheat sheet if needed
